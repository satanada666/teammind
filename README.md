# ⚡ TeamMind — AI ассистент для команды

## Быстрый старт (Windows)

1. Установи Node.js: https://nodejs.org (кнопка "LTS", просто Next → Next → Install)
2. Дважды кликни **ЗАПУСТИТЬ.bat**
3. Введи API ключ когда спросит
4. Открой браузер: http://localhost:3000

---

## Быстрый старт (Mac / Linux)

```bash
node server.js
# или с ключом сразу:
ANTHROPIC_API_KEY="sk-ant-api03-..." node server.js
```

---

## Где взять API ключ

1. Зайди на https://console.anthropic.com/settings/keys
2. Нажми "Create Key"
3. Скопируй ключ (начинается с `sk-ant-api03-...`)

## Почему нужен сервер?

Браузер не может напрямую обращаться к Anthropic API из-за CORS.
Сервер работает как прокси: браузер → localhost:3000 → Anthropic API.
API ключ хранится только на сервере, в браузер не передаётся.
